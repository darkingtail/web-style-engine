# Phase 5: SSR and Hydration

[Back to roadmap](./ROADMAP.md)

## Goal

Support production-grade SSR workflows.

## Tasks

- Hydrate existing style tags in DOM renderer.
- Add request-scoped SSR registry examples.
- Preserve extraction order.
- Extract critical CSS.
- Extract CSS vars, globals, keyframes, and font faces.
- Add multi-instance SSR tests.
- Reserve streaming SSR protocol.

## Acceptance

- SSR extraction and client hydration avoid duplicate style insertion.
- Multi-instance SSR examples are covered by tests.
