# Phase 2: Responsive Completion

[Back to roadmap](./ROADMAP.md)

## Goal

Make responsive a complete core capability for websites across devices.

## Tasks

- Add media feature helpers:
  - `print`
  - `colorScheme`
  - `reducedMotion`
  - `contrast`
  - `forcedColors`
- Add or reserve container query helpers.
- Add H5 helpers:
  - `px2vw`
  - safe-area helpers
  - viewport unit helpers
- Add iframe and Shadow DOM responsive observer tests.
- Add a full landing page example:
  - mobile
  - tablet
  - desktop
  - wide

## Acceptance

- Responsive examples demonstrate real website layout changes.
- Media helpers are typed and tested.
- SSR-safe responsive behavior remains stable.
