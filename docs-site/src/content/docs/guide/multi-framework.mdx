---
title: Multi-framework demos
description: Astro can host Vue, React, and Solid demos in the same documentation site.
---

Astro Starlight is used because the project itself is framework-agnostic. The docs site can host Vue, React, Solid, Svelte, Lit, or Web Component examples without making the style engine core framework-specific.

```astro
---
import VueDemo from '../../components/VueDemo.vue'
import ReactDemo from '../../components/ReactDemo.tsx'
import SolidDemo from '../../components/SolidDemo.tsx'
---

<VueDemo client:load />
<ReactDemo client:load />
<SolidDemo client:load />
```
