`web-style-engine` is a framework-agnostic style engine for Web UI.

It provides reusable style primitives extracted from `antdv-style` without binding the core package to Vue, React, Solid, antdv, or any specific design system.

## First slice

The initial implementation covers the six documented steps:

1. Style Engine Core protocol
2. Renderer / Runtime abstraction
3. createStylesCore abstraction
4. Framework adapter protocol
5. Token / CSS variables / transformer integration
6. antdv-style migration validation scaffold
