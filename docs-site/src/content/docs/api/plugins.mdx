`web-style-engine` supports both function transformers and object-style plugins.

```ts
const engine = createStyleEngine({
  transformers: [
    px2rem({ rootValue: 16 }),
    rtlTransformer(),
  ],
  plugins: [
    prefixer({ order: 10 }),
  ],
})
```

## Plugin Protocol

```ts
const plugin = {
  name: 'my-plugin',
  order: 1,
  transform(cssText, context) {
    return cssText
  },
}
```

Plugins run by ascending `order`. Plugins with the same order keep declaration order. Function transformers remain supported and are assigned generated names such as `transformer-0`.

## Diagnostics

```ts
const engine = createStyleEngine({
  diagnostics: true,
  plugins: [prefixer()],
})

engine.css({ display: 'flex' })
engine.getDiagnostics()
engine.clearDiagnostics()
```

Diagnostics include plugin name, input, output, change status, and style options.

## Built-In Helpers

```ts
prefixer()
rtlTransformer()
px2rem({ rootValue: 16, minPixelValue: 1 })
px2vw({ viewportWidth: 375, precision: 4 })
```

`px2rem` and `px2vw` support `include` and `exclude` filters. `rtlTransformer` swaps common physical left/right CSS output. `prefixer` covers focused runtime compatibility cases such as flexbox, `user-select`, `appearance`, and `backdrop-filter`.
