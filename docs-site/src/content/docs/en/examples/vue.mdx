This example uses the Vue-facing adapter from the main `web-style-engine` package. The component keeps local Vue state, passes state as style props, and uses the shared responsive helper through a composable-style facade.

<VueFullExample client:only="vue" />

## Key imports

```ts
import {
  createDOMRenderer,
  createResponsive,
  createStyleEngine,
  createVueStyleSystem,
} from 'web-style-engine'
```

## Pattern

```ts
const system = createVueStyleSystem({
  engine,
  theme,
  responsive,
})

const useStyles = system.createUseStyles(({ theme, responsive }, props) => ({
  root: {
    color: theme.text,
    ...responsive.object({
      up: {
        regular: { gridTemplateColumns: '1fr auto' },
      },
    }),
  },
}))
```

For a Vue app, provide `system.StyleProvider.getValue()` from your application shell and wrap `system.createUseStyles()` in your own composable. The package itself does not import Vue.
