`web-style-engine` is a framework-agnostic enterprise-grade Web style engine.

It provides reusable style primitives for enterprise component libraries, design systems, responsive official websites, and complex Web applications without binding the core package to Vue, React, Solid, antdv, or any specific design system.

## Modules

- Core style engine
- Runtime renderers
- Responsive query and observer helpers
- createStylesCore
- Framework adapters
- Token and CSS variable helpers
