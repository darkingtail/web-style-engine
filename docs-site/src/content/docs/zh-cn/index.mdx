`web-style-engine` 是一个与框架无关的企业级 Web 样式引擎。

它为企业级组件库、设计系统、响应式官网和复杂 Web 应用提供可复用的样式基础能力，同时不会把核心包绑定到 Vue、React、Solid、antdv 或任何特定设计系统。

## 模块

- 样式引擎核心
- 运行时渲染器
- 响应式查询与观察器
- createStylesCore
- 框架适配器
- Token 与 CSS 变量工具
