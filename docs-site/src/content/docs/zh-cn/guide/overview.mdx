title: 概览
description: "web-style-engine 的定位与首批能力。"
---

`web-style-engine` 是一个与框架无关的企业级 Web 样式引擎。

它提供可复用的样式基础能力，服务于企业级组件库、设计系统、响应式官网和复杂 Web 应用，同时核心包不绑定 Vue、React、Solid、antdv 或任何特定设计系统。

## 首批能力

当前实现覆盖这些能力：

1. 样式引擎核心协议
2. Renderer / Runtime 抽象
3. createStylesCore 抽象
4. 框架适配器协议
5. Token / CSS 变量 / transformer 集成
6. 面向上层设计系统包的生态边界策略
